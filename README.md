# tess4j-test
使用tess4j解析简单的验证码    
1.把tessreact项目里面的tessdata文件夹提取出来放在某个位置：https://github.com/tesseract-ocr/tesseract  
2.加入验证码识别的相关语言包：https://github.com/tesseract-ocr/tessdata  
例如你想识别一般的英文数字验证码，就下 eng.traineddata，然后放到tessdata文件夹下
